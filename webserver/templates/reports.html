This page provides a detailed analysis of a specific log bundle. Hereâ€™s how to use it:

  • Tabs: Switch between Histogram, Table View, GFlags, Node Info, Log Details, Warnings, and Related Reports for different insights.
  • Histogram: Visualize log event frequency over time. By default, It displays the histogram for the last 7 days from the last timestamp in the report. Use filters for node, process, interval, and time range. Toggle log scale for better visibility of spikes.
  • Table View: See a breakdown of log messages, their counts, and first/last occurrence.
  • GFlags: Review the configuration flags used by each process at the time of the log bundle.
  • Node Info: Get information about each node in the cluster, including hardware and software details.
  • Log Details: See parsed log lines and details or solutions if available.
  • Warnings: Review any detected issues, anomalies, or recommendations for the cluster.
  • Related Reports: Quickly jump to other reports from the same cluster, org.
Tip: Use the filters above the histogram to focus on specific nodes, processes, or time intervals. Hover over chart points for details.
